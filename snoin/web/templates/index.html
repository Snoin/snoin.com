Services

Snoin은 다음과 같은 서비스를 제공합니다.

웹 서비스 개발

Snoin은 웹 개발을 주로 담당하는 개발 팀입니다. 전세계의 수많은 스타트업들이 사용하고 있는 최신 버전의 Python과 Flask를 사용하여 서비스를 개발합니다.

반응형 웹

Snoin은 어떻게 하면 여러가지 환경에서 웹 서비스가 사용자에게 정보를 잘 전달할 수 있을지 생각합니다. CSS3과 JavaScript를 사용하여 반응성을 극대화 시킵니다.

웹 표준과 시멘틱

Snoin은 웹 표준을 중요시합니다. HTML5 표준을 준수하며, 문서의 시멘틱(Sementic)을 이뤄서 보다 검색엔진 등의 타 서비스 연동에 친화적인 서비스를 개발합니다.
