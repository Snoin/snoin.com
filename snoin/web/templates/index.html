Features

Snoin은 이런 일을 합니다.

웹 서비스 개발

IoT(사물인터넷:Internet of Things) 시대에 웹 보다 범용적인 접근 수단은 없습니다.

Snoin은 전 세계 수 많은 기업들이 채택하고, 탁월한 생산성이 입증된 최신 버전의 프로그래밍 언어와 개발 프레임워크 등을 사용하여 범용적이고 안정적인 서비스를 보다 빠르게 개발합니다.

모바일 사이트와 반응형 웹

모바일 시대가 열렸지만 아직도 모바일 페이지를 그저 "작은 웹 페이지" 정도로 여기는 회사들이 많습니다.

Snoin은 스마트폰 등 다양한 기기와의 호환성을 극대화 시키는 반응형 웹으로 개발하여, 사이트 방문자들로 하여금 보다 편리한 사용자 경험(UX)를 제공합니다.

웹 표준과 시멘틱

기기들이 다양해지는 시대에 표준을 지키지 않은 사이트는 살아남을 수 없습니다.

Snoin은 사이트의 웹 표준을 준수하며, 시멘틱 구조(Sementic Structure)를 중요시 합니다. 이미 구글 등 세계적인 사이트들은 체계적인 시멘틱 구조 문서를 보다 상위에 노출하고 있습니다.

보안

과거의 웹 에이전시들은 비밀번호 등의 위험한 정보를 아예 암호화하지 않거나, 취약한 방법을 사용해 문제가 발생했습니다.

Snoin은 기본적으로 보안 전문가들이 인증한 방식으로 개인정보를 다루며, 국내법에서 암호화해서 취급하도록 되어있는 정보들 또한 안전하게 취급할 수 있습니다.

품질 관리

대부분의 웹 에이전시들은 사이트 개발을 빨리 끝내는 것이 목적일 뿐, 사이트 자체의 품질은 책임지지 않습니다.

Snoin은 버전관리 시스템을 이용해서 어플리케이션을 관리하며, 질 높고 안정적인 서비스 제공에 힘쓰고 있습니다.

최적화 컨설팅

Snoin은 단지 웹 사이트를 개발하는 데에 그치지 않습니다.

저희는 분산 서비스 거부 공격(DDoS Attack)등에 대비하는 전략, 사이트의 고객들에게 보다 나은 웹 서핑 속도와 보다 안전한 웹 이용을 위한 전략을 제안해드립니다.

About

Snoin은 이런 팀입니다.

  • 2004 – 2008

    꺄울!

    Snoin은 꺄울이란 이름으로 시작되었습니다.
    음악 라디오 서비스인 꺄울캐스트와 호스팅 서비스를 운영했습니다.

  • 2008 –

    Snoin!

    새로운 서비스를 시작하고자 팀 이름을 Snoin으로 바꾸었습니다.
    스노인은 눈사람을 의미하며,
    정성스레 눈사람을 만들던 추억처럼
    웹서비스를 정성들여 만들겠다는 뜻이 담겨있습니다.

  • 2008 – 2015

    Snoin File Storage

    Snoin이 선보였던 웹 클라우드 서비스 입니다!
    이미지와 영상 등 다양한 컨텐츠 보관 서비스를 제공했습니다.

  • 2015 가을

    웹 서비스 제작 서비스 시작

    오랜 기간 서비스를 직접 기획·개발하고 운영했던 경험으로
    Snoin이 이제 고객님의 웹서비스를 완벽하게 제작해드립니다.
